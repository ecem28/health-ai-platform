# 🧠 HealthAI Platform – Final Full Integration

## 🚀 Project Overview

HealthAI is a web-based collaboration platform designed to connect **Engineers** and **Healthcare Professionals** to co-create innovative healthcare solutions.

The platform enables structured collaboration through:

* Project sharing
* Role-based interaction
* Meeting workflows
* Administrative monitoring and control

---

## 🧩 Key Features

### 🔐 Authentication & Security

* JWT-based authentication system
* Role-based access control (Engineer / Healthcare Professional / Admin)

### 👥 User Roles

* Engineer
* Healthcare Professional
* Admin (System Management)

### 📝 Core Functionalities

* Project (Post) creation and management
* Advanced search and filtering system
* Meeting request and scheduling module

### 🛡️ Admin Panel

* User management (activate / suspend accounts)
* Post moderation (delete inappropriate content)
* Audit log monitoring

### 📊 Audit Logging System

* Tracks system activities
* Filterable by date and action type
* Export logs as CSV

### 🎨 UI / UX

* Light / Dark mode toggle (🌞 / 🌙)
* Modern glassmorphism UI
* Responsive design

---

## 🏗️ Tech Stack

### Frontend

* React.js (Vite)
* Context API (State Management)
* CSS Variables (Dynamic Theming)

### Backend

* Node.js
* Express.js
* RESTful API Architecture

### Database

* SQLite

### DevOps

* Docker
* Docker Compose
* Nginx

---

## 🐳 Running the Project

Make sure Docker is installed, then run:

```bash
docker-compose up -d --build
```

### 🔗 Access Points

* Frontend → http://localhost:5173
* Backend → http://localhost:5000
