# 🧠 HealthAI Platform – Final Full Integration (Frontend + Backend + Docker)

This is the **final integrated version** of the HealthAI platform, combining frontend, backend, and containerized deployment into a complete system.

---

## 🚀 Project Overview

HealthAI is a web-based collaboration platform that connects **Engineers** and **Healthcare Professionals** to co-create innovative healthcare solutions in a secure and structured environment.

The system enables interdisciplinary collaboration through project sharing, communication workflows, and controlled access.

---

## 🧩 Features

* 🔐 JWT-based Authentication
* 👥 Role-based system (Engineer / Healthcare Professional / Admin)
* 📝 Project (Post) creation and management
* 🔎 Advanced search and filtering
* 📅 Meeting request and scheduling system
* 🛡️ Admin panel (user management & activity control)
* 📊 Audit logging system (tracking all critical actions)

---

## 🏗️ Tech Stack

### Frontend

* React.js (Vite)
* Context API (State Management)
* Custom CSS (Glassmorphism UI, animations)

### Backend

* Node.js
* Express.js
* RESTful API Architecture

### Database

* SQLite (lightweight relational database)

### DevOps & Deployment

* Docker & Docker Compose
* Nginx (Reverse Proxy)
* Multi-container architecture

---

## 🐳 Running the Project

Make sure Docker is installed, then run:

```bash
docker-compose up -d --build
```

### Access Points

* Frontend → http://localhost:5173
* Backend → http://localhost:5000
